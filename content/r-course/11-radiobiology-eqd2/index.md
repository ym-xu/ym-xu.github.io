放疗研究里几乎总会碰到一个问题:**45 Gy 分 25 次** 和 **6 Gy 一次** 的近距离照射,谁的"生物效应"更大?物理剂量不能直接比——分次方式不同,同样的总剂量对组织的损伤天差地别。把它们换算到**等效 2 Gy 分次剂量(EQD2)** 才能加、能比、能放进同一个模型。

这正是你两篇工作里反复在做的事:[食管癌病例系列](../) 的 Table 1 有 EQD2 列,子宫内膜癌 poster 报的是 D90 的 EQD2。现在多半靠手算或 Excel——这一课把它写成**几行可复用的 R 函数**,顺便补上前面没专门讲过的:**怎么自己写函数 `function()`**。

> 🔬 这一课纯 **base R**,不装任何包,点 Run 立刻出结果。

## 这一课你会做出什么

- 看懂 **线性二次(LQ)模型** 为什么能比较不同分次方案
- 写出自己的 `bed()` 和 `eqd2()` 函数,并用已知例子验证
- 把一张"逐病人剂量表"批量换成 EQD2,**亲手复现你论文 Table 1 的 EQD2 肿瘤列**
- 知道 EBRT + 近距离剂量怎么(以及何时不能)相加

## 1. 为什么不能直接比总剂量

同样 60 Gy,**常规分次**(2 Gy×30)和**大分次**(6 Gy×10)对**晚反应组织**(脊髓、黏膜纤维化)的伤害完全不同:每次剂量越大,晚反应损伤越重。线性二次模型用一个组织参数 **α/β** 来刻画这种"对分次剂量的敏感度":

| α/β(Gy) | 典型组织 | 含义 |
|---------|---------|------|
| ~10 | 多数**肿瘤**、急性反应组织(黏膜炎、皮肤) | 对分次大小不敏感 |
| ~3 | **晚反应**正常组织(脊髓、肺纤维化、食管狭窄) | 对大分次很敏感,大分次伤害重 |
| ~4.5 | 部分肿瘤的折中取值(你论文里用过) | 介于两者之间 |

> ⚠️ **同一个剂量、不同 α/β,换算结果不同**。算肿瘤控制用 α/β≈10,算晚期毒性用 α/β≈3。用错组织的 α/β,结论会偏。

## 2. 线性二次模型 → BED:写第一个函数

**BED(生物等效剂量)** 把"总剂量 D、每次剂量 d"折算成一个与分次无关的量:

> **BED** = `D × (1 + d / (α/β))`

其中 D = 总剂量 = n×d(n 次,每次 d Gy)。我们把它写成一个**函数**——这样以后只要调用 `bed(...)`,不用每次重抄公式:

```r
# 自己写函数:function(参数) { 函数体;最后一行的值就是返回值 }
bed <- function(D, d, ab) {
  D * (1 + d / ab)        # ab 就是 α/β
}

# 用一下:60 Gy 分 30 次(d=2),晚反应组织 α/β=3
bed(D = 60, d = 2, ab = 3)
```

```r
# 对比:同样 60 Gy 但大分次 6 Gy×10(d=6),晚反应 α/β=3
bed(D = 60, d = 6, ab = 3)   # BED 明显更高 → 对晚反应组织伤害更大
```

两次总剂量都是 60 Gy,但大分次的 BED 高得多——这就是"不能直接比总剂量"的量化证据。

## 3. EQD2:换到"2 Gy 分次"这把通用尺子

BED 的单位不直观(它是"无限小分次"的极限)。临床更常用 **EQD2**:"要达到同样效应,用标准 2 Gy/次 需要多少 Gy"。公式是 BED 再除以一个因子,化简后:

> **EQD2** = `D × (d + α/β) / (2 + α/β)`

```r
eqd2 <- function(D, d, ab) {
  D * (d + ab) / (2 + ab)
}

# 验证:45 Gy 分 25 次(d=1.8),肿瘤 α/β=10
eqd2(D = 45, d = 1.8, ab = 10)   # 应 ≈ 44.25:每次<2Gy,EQD2 略低于物理剂量,合理
```

```r
# 同一方案换晚反应 α/β=3
eqd2(D = 45, d = 1.8, ab = 3)    # ≈ 43.2
```

> 直觉自查:每次 **<2 Gy** 时 EQD2 略**低于**总剂量;每次 **>2 Gy**(大分次/近距离)时 EQD2 **高于**总剂量。结果方向不对,八成是 D 和 d 传反了。

## 4. 🔬 批量换算:复现你论文 Table 1 的 EQD2 列

真实场景不是算一个方案,而是一整张**逐病人剂量表**。`eqd2()` 是**向量化**的——直接喂一整列就行,不用循环。下面用你食管癌论文 Table 1 里几位病人的 EBRT 方案,**算出 EQD2 肿瘤(α/β=10)并和论文已发表的数字对一下**:

```r
eqd2 <- function(D, d, ab) D * (d + ab) / (2 + ab)

# 论文 Table 1 节选:EBRT 总剂量与分次
pt <- data.frame(
  patient   = c(2, 8, 9),
  ebrt_dose = c(61.2, 50.4, 50),   # 总剂量 Gy
  ebrt_fx   = c(34,   28,   28)     # 分次数
)
pt$ebrt_d  <- pt$ebrt_dose / pt$ebrt_fx                 # 每次剂量 d
pt$eqd2_t  <- round(eqd2(pt$ebrt_dose, pt$ebrt_d, 10), 1)  # 肿瘤 EQD2
pt
```

再把每位病人的**近距离(EBT)**部分也换算并相加(下一节讲为什么能加):

```r
eqd2 <- function(D, d, ab) D * (d + ab) / (2 + ab)

pt <- data.frame(
  ebrt_dose = c(61.2, 50.4, 50),
  ebrt_fx   = c(34,   28,   28),
  ebt_dose  = c(10,   10,   15),    # 近距离总剂量
  ebt_fx    = c(2,    2,    3)
)
ebrt_eqd2 <- eqd2(pt$ebrt_dose, pt$ebrt_dose / pt$ebrt_fx, 10)
ebt_eqd2  <- eqd2(pt$ebt_dose,  pt$ebt_dose  / pt$ebt_fx,  10)

data.frame(
  total_eqd2 = round(ebrt_eqd2 + ebt_eqd2, 1),
  published  = c(72.7, 62.1, 67.9)   # 论文 Table 1 的 EQD2 肿瘤列(病人 2/8/9)
)
```

三个数应当和论文**完全对上**(72.7 / 62.1 / 67.9)。这就是把"手算/Excel"搬进 R 的意义:**一个函数、可复现、改一个剂量整张表自动更新**,还能贴进 Quarto 报告([L13](../13-capstone/))。

## 5. EBRT + 近距离能直接相加吗?

上面我们把两段的 EQD2 **加**起来了。这在文献里是常规做法,但**前提**要清楚,否则会出错:

- ✅ **同一组织、同一 α/β**:两段都换成"肿瘤 α/β=10"的 EQD2 再加,才是同一把尺子上的量。
- ⚠️ **局部叠加近似**:EQD2 相加假设两段照射打的是**同一块组织**。EBRT 是大野、近距离是陡梯度的小野,真正重叠的体积有限——所以"总 EQD2"是个**简化指标**,不是某一点的真实生物剂量。
- ⚠️ **黏膜/晚反应点的剂量 ≠ 处方剂量**。近距离的剂量梯度极陡,黏膜表面受量常远高于处方点——这就是为什么你论文里 **EQD2 黏膜(α/β=3)列比肿瘤列高很多**,而不是简单换个 α/β 就能从处方剂量推出来(它用的是黏膜点的实际受量)。换算公式不变,**喂进去的 D 要是该组织那一点的剂量**。

> 🔬 一句话:`eqd2()` 只负责"分次换算";"该用哪个剂量、哪个 α/β、能不能加",是放射生物学判断,函数替你算不了。

## 6. 接回统计分析

算出 EQD2 之后,它就是一个普通的**连续变量**,可以接着进前面的分析:

- 当协变量进 [逻辑回归 L7](../07-logistic-regression/) / [Cox L9](../09-survival/) 时,**用 EQD2 而不是物理剂量**——不同分次方案的病人才有可比性。
- 想看"剂量-毒性"关系,EQD2 是比总剂量更合理的横轴([L8 线性回归](../08-linear-regression/) / ROC 找阈值见 [L7](../07-logistic-regression/))。
- ⚠️ 但你的队列很小(n≈11),把 EQD2 粗暴二分成"高/低"或硬塞进多因素模型都很危险——见下一课 [L12 极小样本的诚实分析](../12-small-samples/)。

## 小结

| 你想要 | 公式 / 函数 |
|--------|------------|
| 生物等效剂量 | `bed(D, d, ab) = D*(1 + d/ab)` |
| 等效 2 Gy 分次剂量 | `eqd2(D, d, ab) = D*(d+ab)/(2+ab)` |
| 肿瘤控制 | α/β ≈ 10 |
| 晚反应正常组织 | α/β ≈ 3 |
| 批量换算一张表 | 函数向量化,直接喂整列,不用循环 |
| EBRT+近距离合并 | 同 α/β 的 EQD2 相加(局部叠加近似) |

## 常见误解 Common misconceptions

- "总剂量一样,生物效应就一样" → 错。分次不同,EQD2 差很多,见第 2 节。
- "算毒性也用 α/β=10" → 错。晚反应组织用 α/β≈3,用错会**低估**大分次的晚期损伤。
- "把 EBRT 和近距离的物理剂量直接相加" → 错。要先各自换成同 α/β 的 EQD2 再加,见第 4–5 节。
- "换个 α/β 就能从处方剂量推出黏膜 EQD2" → 错。黏膜受量 ≠ 处方剂量(陡梯度),见第 5 节。

## 常见报错 Troubleshooting

- 结果方向反了(每次>2Gy 却 EQD2 偏低) → `D`(总剂量)和 `d`(每次剂量)传反了。
- 复现不出论文的数 → 先确认 `d = 总剂量 / 分次数` 算对了,以及用的是肿瘤 α/β=10。
- `Error: object 'eqd2' not found` → 函数定义和调用要在**同一个**代码块里跑(每个 Run 是独立环境)。
- 想对一整张 Excel 表换算 → 用 [L2 数据整理](../02-data-wrangling/) 把它读成 data.frame,列名对上后整列喂进 `eqd2()`。
